# logs — 실행 이력 (Evaluate 증거)

이 폴더에는 `docs/rules/workflow.md`의 "로그" 절 규칙에 따라 **기능별 실행 로그**가 쌓입니다.

- 파일 하나 = 기능 하나: `docs/logs/{기능}.md` (예: `point-charge.md`, `order.md`, `concurrency-test.md`)
- 작성법·템플릿은 [`docs/rules/workflow.md`](../rules/workflow.md)의 "로그" 절 참고.

아직 구현 전이라 비어 있습니다. 각 기능의 Evaluate 단계에서 첫 로그 파일이 생성됩니다.
